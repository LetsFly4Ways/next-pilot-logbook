# Privacy Statement

**Last updated:** 01 February 2026

## Overview

Next Pilot Logbook is an open-source web application designed to store and manage pilot logbook–related information. We take privacy seriously and aim to be transparent about what data is collected, how it is used, and where it is stored.

## Information We Collect

Next Pilot Logbook may collect and store the following information, depending on how the application is used and configured:

### Personal information

- First name and last name
- Email address
- Phone number
- License or certification number
- Company or organization name

### Logbook data

- Logbook entries and flight records
- Dates, times, aircraft details, notes, and other content entered by users
- Any additional information users choose to include in logbook items

### Technical information

- Data necessary for the application to function (such as authentication data, timestamps, and system logs)

## How the Information Is Used

The collected information is used solely to:

- Provide and operate Next Pilot Logbook’s core functionality
- Associate logbook entries with the correct user
- Enable authentication and account management
- Maintain data integrity and system security

Next Pilot Logbook does **not** use personal data for advertising, marketing, or tracking.

## Data Storage and Access

All data is stored in the database of the Next Pilot Logbook instance where the application is deployed.

- If you self-host Next Pilot Logbook, you are fully responsible for the storage, security, and management of your data.
- The maintainers of the Next Pilot Logbook open-source project do **not** have access to user data stored in self-hosted or third-party deployments.

## Data Sharing

Next Pilot Logbook does not sell, rent, or share personal data with third parties.

Data may be shared only if:

- Required by law
- Explicitly configured by the operator of a specific deployment (for example, integrating third-party services)

## Cookies

Next Pilot Logbook may use cookies strictly necessary for functionality, such as maintaining user sessions.  
Next Pilot Logbook does not use tracking or advertising cookies.

## Data Retention

Data is retained for as long as it remains in the Next Pilot Logbook database. Users or system administrators may delete data at any time, depending on deployment configuration.

## Security

Reasonable technical measures are taken to protect stored data from unauthorized access, loss, or misuse. However, no system can be guaranteed to be 100% secure.

## Open Source Transparency

Next Pilot Logbook is open source. The publicly available source code allows anyone to inspect how data is handled and stored.

## Your Rights

Depending on your jurisdiction, you may have the right to:

- Access your personal data
- Request correction of inaccurate data
- Request deletion of your data

Requests should be directed to the administrator or operator of the Next Pilot Logbook instance you are using.

## Changes

This Privacy Statement may be updated from time to time. Any changes will be reflected on this page with an updated revision date.

## Contact

For questions related to privacy and the Next Pilot Logbook project itself, please contact the project maintainer via the source code repository.
